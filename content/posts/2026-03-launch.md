We are excited to announce **BAM Engine 0.5.1** and the launch of
[bamengine.org](https://bamengine.org).

<!--more-->

## What is BAM Engine?

BAM Engine is a Python framework implementing the BAM (Bottom-Up Adaptive
Macroeconomics) agent-based model from *Macroeconomics from the Bottom-up*
(Delli Gatti et al., 2011). The model simulates households, firms, and banks
interacting across labor, credit, and goods markets.

## What's New in 0.5.1

### Extension Bundles

The new `Extension` dataclass and `sim.use()` API let you activate model
extensions with a single call:

```python
from extensions.rnd import RND
sim.use(RND)
```

Pre-built bundles: R&D/Growth+, buffer-stock consumption, and taxation.

### Documentation

The User Guide now covers custom roles, events, relationships, pipelines,
operations, extensions, validation, calibration, and best practices.
