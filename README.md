# bamengine.org

Source for [bamengine.org](https://bamengine.org), the project website for
[BAM Engine](https://github.com/kganitis/bam-engine).

Built with [Hugo](https://gohugo.io/) and the
[scientific-python-hugo-theme](https://github.com/scientific-python/scientific-python-hugo-theme).

## Development

Requires [Hugo extended](https://gohugo.io/installation/).

```bash
hugo server
```

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically.
